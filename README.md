# 🚗 AutoGear - Full Stack E-Commerce Website

AutoGear is a full-stack e-commerce web application for car accessories built using **React.js** and **Django REST Framework**. It provides a modern shopping experience with secure authentication, product browsing, wishlist management, shopping cart, and order placement.

---

## 📌 Features

- User Registration
- User Login (JWT Authentication)
- Secure Protected Routes
- Product Listing
- Product Details
- Category-wise Products
- Search Products
- Shopping Cart
- Wishlist
- Quantity Management
- Order Placement
- Responsive User Interface

---

## 🛠️ Technologies Used

### Frontend
- React.js
- React Router DOM
- Axios
- Context API
- CSS

### Backend
- Django
- Django REST Framework
- Simple JWT Authentication
- SQLite
